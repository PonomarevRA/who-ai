# Инструкции для ИИ-агентов

Это автономный Vite-проект на чистом JavaScript. Он должен работать двумя способами: через Vite и при прямом открытии `index.html` по `file://`.

## Архитектура

- `src/mcp/animalIdentity.js` — единственный источник данных и правил: 300 животных, цветов и прилагательных, SVG, ежедневный прогноз и хэши.
- `src/main.js` — DOM, поиск, анимация, кнопки копирования и безопасная ссылка «поделиться».
- `app.js` — собранный классический бандл для `file://`. Не редактируйте вручную: генерируйте через `npm run build:direct`.
- `favicon.svg` — иконка сайта; сохраняйте её векторной и без внешних зависимостей.
- `llms.txt` — публичная инструкция для ChatGPT и других ассистентов. Обновляйте её при изменении сценариев, приватности или назначения сайта.

## Правила изменений

- Не передавайте введённый пользователем текст в URL, аналитику или внешние запросы. Ссылки «поделиться» могут содержать только идентификаторы животного, цвета и варианта.
- В Яндекс Метрику передавайте только заранее заданные имена UI-событий. Не добавляйте в неё текст поиска, результаты генерации, URL c query/hash или содержимое промптов.
- Сохраняйте детерминированность: животное и цвет считаются из `текст + текущий месяц`; прогноз — из `текст + дата + период до/после 16:00` по Москве.
- Для каждого нового животного добавляйте узнаваемую SVG-деталь, сцену и уникальный слоган.
- После изменений запускайте `npm run validate` и `npm run build`.
